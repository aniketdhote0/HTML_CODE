Emphasized

The content of an em element is intended to be a point of emphasis, and it is usually dispayed in italicized text.


Strong

The Strong Element is intended to show strong emphasis for its content; stronger emphasis than the em element.


Abbreviation

Can indicate an abbreviated form by placing the abbreviation between opening abbr and closing /abbr tags.


Acronym

The Acronym indicates that the text between an opening acronym and dosing /acronym element is an acronym.


Definition

This tutorial teaches how mark up your documents for the web using XHTML


Blockquote

When we want to quote a passage from another source, we should use the < blockquote > element. Text inside a < blockquote > element is usually indented from the left and right edges of the surrounding text, and sometimes uses an italicized font

EX:-
XHTML 1.0 is the W3C's first Recommendation for XHTML, following on from earlier on HTML 4.01, HTML 4.0, HTML 3.2 and HTML 2.0.

Quotations

The < q > is intended to be used when you want to add quotes (“ “) within a sentence rather than as an indented block on its own.


Citations

If we are quoting a text, we can indicate the source placing it between an opening < cite > tag and closing < /cite > tag. The content of the < cite > element is rendered in italicized text by default.


computer code

Any code to appear on a Web page should be placed inside a < code > element. Usually the content of the < code > element is presented in a monospaced font, just like the code in most programming books


Keybord text

If we want to tell a reader to enter some text, you can use the < kbd > element to indicate what should be typed in. The content of a < kbd > element is usually represented in a monospaced font like the content of the < code > element

EX:-

ctrl + alt


Variables

This element is usually used in conjunction with the < pre > and < code > elements to indicate that the content of the element is a variable that can be supplied by a user

EX:-

Document.write( " user-name ")


Samp

The < samp > Element indicates sample output from a program,script of the like it is mainly used when documentring programming concepts

EX:-

Result produced by the program is Hello world


Address

The < address > Element is used to contain any address it is displayed in italicised

EX:-
shivajinagar, chandrapur-(M.S.) INDIA,442401
