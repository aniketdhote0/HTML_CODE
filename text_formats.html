HTML TAGS

Html Tag

The main container for HTML pages

EX:-  
	<*html>
	<*head>
	<*meta charset="utf-8">
	<*meta name="viewport" content="width=device-width, initial-scale=1">
	<*title>WORLD OF HTML<*/title>
<*/head>
<*/html>

Head Tag

The container for page header information (meta data)

EX:-  
	<*head>
	<*meta charset="utf-8">
	<*meta name="viewport" content="width=device-width, initial-scale=1">
	<*title>WORLD OF HTML<*/title>
<*/head>

Title Tag

The title of the page

 Ex:-
		<*head>
			<*title> HTML basic tags <*/title>
		<*/head>
